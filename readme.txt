Display Rank Math FAQ schema data as a customizable accordion on your website.

== Description ==

This plugin fetches FAQ data stored by Rank Math in the post meta and displays it as a stylish, collapsible accordion.

Features:
* Show FAQ items in an accordion UI
* Shortcode [faq_accordion] to insert anywhere in posts or pages
* Admin settings page to customize colors, fonts, and sizes
* Lightweight and optimized for performance

== Installation ==

1. Upload the plugin folder to the /wp-content/plugins/ directory or install via WordPress admin plugin upload.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings » FAQ Accordion to adjust styles and options.
4. Use the shortcode [faq_accordion] in your posts or pages to display the FAQ accordion.

== Frequently Asked Questions ==

= Does this plugin require Rank Math? =
Yes, this plugin displays FAQ data saved by Rank Math's schema feature.

= How do I add FAQs? =
Add FAQ data in Rank Math’s Schema tab when editing posts or pages. Then insert the shortcode to show the accordion.
